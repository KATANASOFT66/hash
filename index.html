ℹ️ How This Obfuscator Works
Layer 1 — Decimal Escape Sequences:
All strings are encoded as \068\090\107... (Lua decimal escapes → ASCII chars)
Layer 2 — Custom Base64 Alphabet:
Strings in the S table are encoded with a shuffled Base64-like encoding using a custom character set
Layer 3 — Bytecode VM:
The script runs on a custom Lua virtual machine with its own instruction set. Full decompilation requires executing the bytecode. This decoder extracts all readable strings and decodes all static layers.
